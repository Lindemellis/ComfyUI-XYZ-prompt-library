# 图片浏览器

浏览和管理 ComfyUI 输出图片。自动索引、标签管理、批量操作、元数据查看。

## 打开

ComfyUI 顶部栏的 **Gallery** 按钮，或直接访问 `/xyz/gallery`。

首次打开时，浏览器会自动扫描 `output` 和 `input` 目录下的图片。

## 浏览图片

### 筛选

左侧面板提供多种筛选器：

| 筛选器 | 说明 |
|---|---|
| 文件夹 | 选择文件夹，可选包含子文件夹 |
| 收藏 | 仅显示收藏的图片 |
| 标签 | 包含指定标签的图片（可多个） |
| 模型 | 按 checkpoint 模型筛选 |
| 提示词 | 搜索提示词中的关键词 |
| 日期范围 | 按文件修改时间筛选 |

### 排序

按时间、文件名、文件大小、文件夹排序，升序或降序。

### 视图模式

- **网格视图**：缩略图卡片，适合快速浏览
- **紧凑视图**：密集缩略图，适合大量图片
- **行视图**：按大小/日期/首字母分组
- **详情视图**：单张图片完整信息

## 标签管理

- 图片详情页可以添加/删除标签
- 批量选中图片后批量添加/删除标签
- Settings → Tag management 可以重命名、删除、清理标签

## 批量操作

1. 选中多张图片
2. 选择操作：移动、删除、收藏、添加标签
3. 系统先执行预检（检查磁盘空间、文件名冲突等）
4. 确认后执行

## 设置

Gallery 页面内的 Settings 面板：

- **主题**：深色 / 浅色
- **下载方式**：原图 / 含元数据
- **筛选面板**：选择哪些筛选器显示
- **开发者模式**：显示额外调试信息

## 文件夹管理

- 默认包含 ComfyUI 的 `output` 和 `input` 目录
- 可以添加自定义文件夹
- 支持创建子文件夹、重命名、移动

## 常见问题

**图片没有自动出现？** 检查文件夹设置，或手动点击 Rescan。

**缩略图加载慢？** 首次访问需要生成缩略图，之后有缓存会很快。

**标签写回图片文件？** 收藏和标签会自动写回 PNG 文件的元数据 chunk。
