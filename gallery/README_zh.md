# 图片画廊（Image Gallery）

[English](README.md) | **中文** · [← 返回主 README](../README_zh.md)

浏览与管理你的 ComfyUI 图片——自动索引，支持筛选、打标签、批量操作和元数据查看。

## 打开

点击 ComfyUI 顶栏的 **Open XYZ Gallery**（图片图标）。首次打开时会扫描你的 `output` 和 `input` 目录，并在后台建立缩略图索引。

## 浏览

### 筛选（左侧面板）

| 筛选 | 说明 |
|---|---|
| 文件夹 | 选择文件夹，可选是否包含子文件夹 |
| 收藏 | 只看已收藏的图片 |
| 标签 | 带有指定标签的图片（从你的库自动补全） |
| 模型 | 按 checkpoint 模型筛选 |
| 提示词 | 在提示词里做关键词搜索 |
| 日期范围 | 按文件修改时间筛选 |

可按时间、文件名、大小或文件夹排序（升序 / 降序）。

### 视图模式

- **Grid（网格）** —— 缩略图卡片。
- **Compact（紧凑）** —— 图多时的密集缩略图。
- **Line（行）** —— 分组行（按大小 / 日期 / 首字母）。
- **Detail（详情）** —— 单张图片的完整元数据（点击图片打开）。

## 标签

- 在详情页增删标签。
- 对选中的图片批量打标签。
- 收藏和标签会镜像写入 PNG 的元数据，随文件一起携带。

## 批量操作

1. 选中多张图片。
2. 在批量栏选择操作：移动、删除、收藏或打标签。
3. 画廊会做预检（如命名冲突）并显示进度。
4. 确认后执行。

## 设置

在画廊的 **Settings** 视图：

- **Theme（主题）** —— 深色 / 浅色。
- **Download（下载）** —— 通过下载选择器决定下载时包含什么（如带元数据）。
- **Filters（筛选项）** —— 选择显示哪些筛选控件。
- **Developer mode（开发者模式）** —— 额外的调试信息。

## 文件夹

- 默认根目录是 ComfyUI 的 `output` 和 `input`。
- 可添加自定义文件夹；在树中创建子文件夹、重命名、移动。

## 常见问题

**图片不显示？** 检查文件夹设置，或触发一次重新扫描。

**缩略图一开始很慢？** 首次查看时生成，之后缓存为 `.webp`。

**数据在哪里？** 在 `gallery_data/`（已 gitignore）：索引数据库和缩略图缓存。除了收藏/标签的元数据镜像外，原始图片不会被修改。
