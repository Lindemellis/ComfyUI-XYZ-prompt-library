# Prompt Library V2

分层提示词库，支持模板、引用语法和触发器别名。在 ComfyUI 内以浮动窗口编辑。

## 基本用法

1. 添加 **XYZ Prompt Library V2 Positive**（正面提示词）或 **Negative**（负面提示词）节点到工作流
2. 点击节点上的 **Library** 按钮打开提示词库窗口
3. 点击节点上的 **Editor** 按钮打开模板编辑器
4. 在编辑器中输入模板文本，节点执行时自动解析

### 节点输入输出

| 端口 | 类型 | 说明 |
|---|---|---|
| `prompt_template` | STRING | 模板文本，支持下方所有语法 |
| `seed` | INT | 控制随机模式的结果 |
| `resolved_prompt` | STRING | 解析后的最终输出 |
| `raw_template` | STRING | 原始模板文本（用于链式连接） |

## 模板语法

### `[ref]` — 条目引用

引用库中已有的条目：

```
[角色名]                  ← 用最短唯一名称引用
[角色名.表情]             ← 用点路径引用子条目
```

### `{a|b|c}` — 多选项

配合 ComfyUI 队列的批量次数使用，每次执行依次选择一个：

```
{大师作品|草图|涂鸦}
```

第 1 次执行 → `大师作品`，第 2 次 → `草图`，超出范围 → 空。

### `(text:1.3)` — 权重

当库中提示词的权重不为 1.0 时自动包装为 `(文本:权重)` 格式。

### 自动清理

解析后自动：
- 合并重复逗号（`,,` → `,`）
- 去除首尾多余定界符
- 压缩多余空格

## 提示词库窗口

### 文件夹树

- 创建文件夹和条目
- 拖拽排序
- 右键菜单：重命名、移动、删除、新建
- 过滤器：正面 / 负面 / 全部
- 折叠/展开全部

### 条目编辑

- 名称和极性（positive / negative / both）
- 触发器别名 — 用于 `[ref]` 引用的快捷名称
- 提示词列表：添加、编辑、拖拽排序
- 随机模式：不随机 / 选择 N 条 / 随机丢弃
- 提示词格式：`{prompt}` 或自定义模板
- 提示词间定界符

### 子条目

条目可以有子条目。子条目的提示词会被继承，`_neg` 子条目会自动插入到负面输出中。

## 文本编辑器窗口

- **单面板 / 分屏**：标签页切换或上下分屏同时编辑正面和负面
- **语法高亮**：有效的 `[ref]` 紫色背景，无效的红色波浪线
- **撤销/重做**：每面板独立
- **查找/替换**：支持大小写、全词匹配
- **智能插入**：分析光标上下文，自动加逗号分隔
- **右键菜单**：添加到条目、创建条目、在详情中打开

## 规范化设置

在 **XYZ Prompt Tools → Insertion** 面板中配置，应用到编辑器中的文本：

| 设置 | 效果 |
|---|---|
| Replace `_` with space | `blue_eyes` → `blue eyes` |
| Auto comma | 插入后自动加 `, ` |
| Escape brackets | `()` → `\(\)`（保留权重语法） |
| Full-width → half-width | `，` → `,` |

## 常见操作

**怎么切换正面/负面编辑？** 编辑器窗口顶部标签栏可切换，或点击分屏按钮同时编辑。

**怎么在编辑器里引用库条目？** 输入 `[` 触发条目的自动补全，或者在树面板右键 → 插入引用。

**怎么保存提示词到库？** 在编辑器里选中文本 → 右键 → 添加到条目 / 创建条目。

**怎么触发继承子条目？** 条目的 `_neg` 子条目会自动插入，不需手动引用。`_template` 子条目定义条目的默认模板。

## 命名规则

- 条目/文件夹名不能包含：`.` `,` `|` `/` `\` `[` `]`
- 触发器名不能包含：空格 `,` `|` `/` `\` `[` `]`
